MI PROYECTO SE BASA EN UN ECOMMERCE DE VENTA DE PRODUCTOS, DONDE LOS USUARIOS PUEDEN COMPRAR Y CREAR SUS PRODUCTOS. DEPENDIENDO DEL ROL QUE TENGAN.
TAMBIÉN EL PROYECTO CUENTA CON UNA PERSISTENCIA EN BASE DE DATOS, LOS CAMBIOS SON EN TIEMPO REAL USANDO SOCKET Y CUENTA CON UNA ARQUITECTURA DE CAPAS SEPARADA POR CARPETAS.
SE IMPLEMENTÓ UN SISTEMA DE PAGOS USANDO PAYPAL PARA LA VENTA DE LOS PRODUCTOS.
EL SISTEMA CUENTA CON UN CHAT EN TIEMPO REAL.
CUENTA CON MODELOS DE USUARIOS,PRODUCTOS,CARRITOS,MENSAJES Y TICKETS DE PAGO.

PARA ESTE PROYECTO SE USÓ EXPRESS JS Y DEMÁS DEPENDECIAS VINCULADAS PARA UN MEJOR FUNCIONAMIENTO DE LA APLICACIÓN

COMANDOS PARA LEVANTAR EL PROYECTO:
npm i
cd src
npx nodemon .\app.js
